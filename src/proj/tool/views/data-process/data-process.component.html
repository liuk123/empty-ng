数据流程化处理 在线工具

思路：从输入框中输入数据，选择方法，上一个方法的输出作为下一个方法的输入，执行一系列的方法，得到输出结果。

功能：数据格式转换、数据处理、数据查找、正则、tree操作等。

格式：输入｛input: 'a'｝、函数体：return v.toUpperCase()、这里说的tree：是数组和对象组成，利用递归遍历操作

可以F12查看每步的输出结果或者报错、可以把配置导出，下次再次使用。
