# InsDemo

## 命令行

### 启动

``` 
// 连接本地环境启动项目
npm run start 
// 连接阿里服务器
npm run start:staging
// 服务器渲染
npm run dev:ssr
```
### 打包

```
npm run build:ssr

```

## 环境配置

``` 
// 是否是生产环境
production: true
// 启用哈希路由
useHash: false
```
